# Wake Word Models

Place your openWakeWord ONNX model files here.

Required files:
- `alexa.onnx` - Wake word model for "Alexa"
- `embedding_model.onnx` - Feature extraction model (from openWakeWord)
- `melspectrogram.onnx` - Audio preprocessing model (from openWakeWord)

You can get pre-trained models from:
https://github.com/dscripka/openWakeWord
